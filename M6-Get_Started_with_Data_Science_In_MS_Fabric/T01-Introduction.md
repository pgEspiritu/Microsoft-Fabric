# 📘 Introduction to Data Science in Microsoft Fabric

## 🎯 Overview

Data science helps organizations make informed decisions by analyzing data, discovering patterns, and building predictive models.

Microsoft Fabric provides a unified environment that supports the **entire data science lifecycle**, allowing teams to manage projects from data preparation to model deployment within a single workspace.

---

# 💡 Why Use Data Science?

Organizations use data science to answer complex business questions and improve decision-making.

### Example Scenarios

- 🛒 Predict how much bread a supermarket should stock each week to reduce waste while meeting customer demand.
- 🎯 Analyze customer behavior to deliver personalized marketing offers.

---

# 🔬 What is Data Science?

**Data Science** is a multidisciplinary field that combines:

- Mathematics
- Statistics
- Computer Engineering
- Programming

It uses data to discover hidden patterns, generate insights, and solve business problems.

---

# 🤖 Data Science and Artificial Intelligence (AI)

One of the primary goals of data science is to build **Artificial Intelligence (AI)** solutions.

A common approach is to:

1. Analyze data.
2. Identify patterns.
3. Train **Machine Learning (ML)** models.
4. Use those models to make predictions or automate decisions.

---

# 🐍 Machine Learning with Python

Data scientists commonly use Python libraries such as:

- **scikit-learn**

to build and train machine learning models.

These models learn from historical data and can make predictions on new data.
