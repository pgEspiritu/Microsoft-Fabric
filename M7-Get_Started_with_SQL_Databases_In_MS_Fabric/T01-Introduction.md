# Introduction to SQL Database in Microsoft Fabric

## Overview

**SQL Database in Microsoft Fabric** is a fully integrated SQL database solution within the Microsoft Fabric ecosystem. It provides a unified platform for managing, querying, and analyzing data while seamlessly integrating with other Microsoft Fabric services.

It supports both **operational (transactional)** workloads and **advanced analytics**, enabling organizations to consolidate their data infrastructure and make better data-driven decisions.

- Supports both transactional and analytical workloads.

# Databases Workload in Fabric

The **Databases workload** is designed to support collaboration among different data professionals:

- **Data Engineers** – Build and manage operational databases.
- **Data Analysts** – Query and analyze business data.
- **Data Scientists** – Access operational data for machine learning and analytics.

The workload enables teams to create, manage, and query data optimized for their specific needs.
