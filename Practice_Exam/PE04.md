# Question 31

### You loaded a Delta table into a PySpark DataFrame:

```python
df = spark.sql("SELECT * FROM Product")
```

You need to display the first **100 rows**.

Options:

- describe(df.limit(100))
- df.describe(100)
- df.printSchema(100)
- ✅ **display(df.limit(100))**

## ✅ Correct Answer

**display(df.limit(100))**

### Explanation

`display()` is the Fabric notebook function that renders a DataFrame in a tabular format.

Example:

```python
display(df.limit(100))
```

Other options:

- `describe()` → Returns summary statistics.
- `printSchema()` → Displays the schema only.

# Question 33

### Which tables should be denormalized to reduce joins?

Options:

- Dimension tables at same granularity
- Fact tables
- Role-playing dimensions
- ✅ **Snowflaked dimension tables**

## ✅ Correct Answer

**Snowflaked dimension tables**

### Explanation

Snowflake schemas normalize dimension tables.

Denormalizing them into a single dimension:

- Reduces joins
- Improves query performance
- Produces a Star Schema

# Question 36

### Sales must exclude records found in ExceptionRecords.

Options:

- Inner Join + Expand
- Left Anti + Expand
- ✅ **Left Anti Join, then remove the Expand column**
- Right Anti Join

## ✅ Correct Answer

**Merge (Left Anti Join), then delete the Expand column**

### Explanation

Left Anti Join returns:

```
Sales
MINUS
ExceptionRecords
```

Since no matching columns are needed afterward, the expanded column should be removed.

This also preserves query folding.

# Question 38

### Save a Delta table partitioned by Year and Quarter.

Options:

- Delta → Files
- ✅ **Delta → Tables**
- Parquet → Files
- Parquet → Tables

## ✅ Correct Answer

```python
df.write.mode("overwrite") \
.format("delta") \
.partitionBy("Year","Quarter") \
.save("Tables/Sales")
```

### Explanation

Managed Delta tables belong in:

```
Tables/
```

Not:

```
Files/
```
